# Shadow Escape Runner 🚀

A fast-paced arcade game where you dodge falling asteroids and rack up your high score.

## Getting Started

Follow these steps to run the game on your own machine.

### 1. Clone the repository

```bash
git clone https://github.com/arun-kavali/ShadowEscape.git
cd ShadowEscape
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npx expo start
```

### 4. Run it on your phone with Expo Go

1. Install the **Expo Go** app from the [App Store](https://apps.apple.com/app/expo-go/id982107779) (iOS) or [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent) (Android).
2. Make sure your phone and computer are on the same Wi-Fi network.
3. Scan the QR code shown in your terminal (or browser window):
   - **iOS**: open the Camera app and scan the code.
   - **Android**: open Expo Go and tap **Scan QR Code**.
4. The game will load on your phone. Any code changes you save will refresh automatically.

## How to Play

- Tap **Start Game** to begin.
- Use the **◀** and **▶** buttons to dodge falling asteroids.
- Survive as long as possible — your score increases each time you avoid an asteroid.
- Your best score is saved automatically and shown on screen.
